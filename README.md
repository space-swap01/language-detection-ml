# language-detection-ml
Language Detection(Romanised Hindi &amp; English ) using Machine Learning

## Project Overview
This project is a machine learning model that detects the language of a given text input as Romanised Hindi or English. 
The model is trained on a dataset containing sentences from English language and Hindi language and predicts the language using text classification techniques.

## Features
- Input text and detect its language
- Trained machine learning model for language classification
- Text preprocessing and feature extraction using TF-IDF

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- TF-IDF Vectorizer
- Logistic Regression

## Dataset
The dataset contains sentences from English languages and Romanized Hindi only and used to train the model.  
Each sentence is labeled with its corresponding language.

## Machine Learning Pipeline
1. Data preprocessing
2. Text cleaning
3. TF-IDF feature extraction
4. Model training
5. Model evaluation
6. Language prediction

## Algorithms Used
- Logistic Regression

## Model Evaluation
The model performance was evaluated using:
- Accuracy score
- Classification report
- Train-test split
